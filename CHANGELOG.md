# Changelog

## v1.2.0 - 2026-06-25

- 찬양 제목이 찬송가와 같을 때 `찬송가 제외` 버튼으로 자동 새찬송가 표기를 끌 수 있게 했습니다.
- 찬양 목록과 헌금/마무리 찬양 모두에서 찬송가 제외 설정이 복사 결과에 반영되도록 했습니다.
- 숫자만 입력한 찬송가 번호에는 찬송가 제외 버튼이 뜨지 않고, 제목 입력으로 매칭될 때만 버튼이 표시되도록 조정했습니다.
- 예배당 디지털 시계에 밀리초 보기 토글을 추가하고, 밀리초 표시 중에도 주요 시간이 흔들리지 않도록 고정 폭으로 정리했습니다.
- 시계 오차 보정에 1초 단위 조정을 추가하고, 보정 버튼을 오차 문구 아래에 `1분 -`, `1초 -`, `초기화`, `1초 +`, `1분 +` 순서로 배치했습니다.

## v1.1.2 - 2026-06-13

- 방송 설명 복사 시 대괄호 섹션 사이에 빈 줄을 하나 더 넣도록 조정했습니다.
- 설명 마지막 섹션 뒤에는 불필요한 추가 줄바꿈이 붙지 않도록 했습니다.
- 시계 화면의 시작 시간 수정 버튼을 우측 상단 닫기 버튼 옆으로 이동했습니다.
- 시작 시간 수정 입력창이 우측 상단 버튼 아래에 열리도록 배치를 변경했습니다.
- 시계 화면 중앙에는 오차 측정 버튼만 남기고, 오차 측정 전 안내 문구를 간결하게 줄였습니다.

## v1.1.1 - 2026-06-10

- 예배당 디지털 시계 오차 측정 시 스페이스바를 누른 순간이 항상 00초 기준으로 맞춰지도록 수정했습니다.
- 오차 값을 초 단위 반올림이 아닌 밀리초 단위로 저장해 59초대에서 시작하는 표시 문제를 해결했습니다.
- 1분 + / 1분 - 보정도 밀리초 기준으로 동작하도록 맞췄습니다.

## v1.1.0 - 2026-06-07

- Copible 원본 흐름을 참고해 성경 본문 자동검색을 개선했습니다.
- 성경책 검색에서 한글 초성/부분 검색과 영문 자판 입력 보정을 지원합니다.
- 예배 정보 영역을 접이식으로 정리하고, 본문/장/절 입력 폭과 숫자 입력 동작을 다듬었습니다.
- 찬양 목록, 헌금, 복사 버튼 배치와 전체 창 폭을 방송용 작업 흐름에 맞게 조정했습니다.
- 버튼과 글씨 색을 무채색 톤으로 정리하고 밝은 배경의 대비를 개선했습니다.
- 예배당 디지털 시계 보기, 오차 측정, 분 단위 보정, 임시 예배 시작 시간 수정 기능을 추가했습니다.
- 카톡 링크 복사 후 DVR 해제 확인 알림을 추가하고, 확인 시 알림이 닫히도록 했습니다.
- 앱 메뉴에 업데이트 확인, GitHub 릴리즈/저장소 열기, 버전 확인, 보기/편집 메뉴를 추가했습니다.
- GitHub 릴리즈 기반 자동 업데이트 설정과 수동 업데이트 오류 안내를 보강했습니다.
- Electron 빌드에서 최신 메인 프로세스 코드가 포함되도록 빌드 스크립트를 수정했습니다.
- 프로그램 실행 창 크기를 실제 패널 폭에 맞게 줄이고 고정했습니다.
